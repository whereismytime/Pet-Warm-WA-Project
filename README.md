# Pet Warm Grooming Website

This project is a static website for a grooming studio called **Pet Warm**, based in Kilkenny, Ireland.

The website was created as part of a web authoring assignment using HTML and CSS. The main idea was to build a simple, clean and easy-to-use website without using JavaScript or any external framework.

## Project Overview

Pet Warm is a small grooming website for dog and cat owners. It gives visitors basic information about services, prices, offers, team members, events and contact details.

The website focuses on:

- clear navigation
- readable layout
- simple page structure
- separate styling files
- basic SEO setup
- user-friendly booking/contact flow

## Website Pages

- `index.html` — home page
- `services.html` — general grooming services
- `services-dogs.html` — dog grooming services
- `services-cats.html` — cat grooming services
- `offers.html` — current offers and discounts
- `masters.html` — grooming team page
- `blog.html` — grooming tips and advice
- `events.html` — workshops and events
- `contact.html` — contact form and studio details
- `thankyou.html` — thank-you page after form submission

## Features

- Static multi-page website
- Dropdown navigation for service pages
- Separate pages for dogs and cats
- Contact form with redirect to a thank-you page
- Hero sections with images
- Service cards and offer cards
- Blog and events sections
- Social media links
- Basic meta descriptions for SEO
- Organised folder structure

## Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Git and GitHub
- FileZilla for uploading the project to hosting
