# Pet Warm Grooming Website

This project is a static website created for a grooming studio called **Pet Warm**, based in Kilkenny, Ireland.  
The website was developed as part of a web authoring assignment using only HTML and CSS.

## Project Overview

The purpose of this website is to present a small business online in a clear and simple way.  
It allows users to explore services, check offers, learn about the team, and contact the studio.

The design focuses on readability, simple navigation, and a clean structure without unnecessary complexity.

## Website Pages

The website includes the following pages:

- **Home (index.html)** – main landing page with introduction and key services  
- **Services (services.html)** – overview of grooming services  
- **Dogs (services-dogs.html)** – dog grooming packages  
- **Cats (services-cats.html)** – cat grooming packages  
- **Offers (offers.html)** – discounts and special deals  
- **Our Masters (masters.html)** – team members  
- **Blog (blog.html)** – grooming tips and pet care advice  
- **Events (events.html)** – workshops and grooming sessions  
- **Contact (contact.html)** – contact form and details  
- **Thank You (thankyou.html)** – confirmation page after form submission  

## Features

- Simple and clear navigation with dropdown menu for services  
- Separate pages for dog and cat grooming  
- Contact form with redirect to a confirmation page  
- Clean layout using reusable CSS styles  
- Images and icons for better visual structure  
- Basic SEO elements (meta descriptions, page titles)  
- External links to social media platforms
